# BoardChat 社区聊天室

一个面向现代社区的全栈项目：将结构化论坛讨论与实时聊天室结合，提供完整的内容发布、互动与治理能力。

## 核心特性

- 论坛模块：发帖、回复、分页与搜索
- 实时聊天：WebSocket 即时通信与重连机制
- 用户中心：个人资料、我发的、我回复的
- 内容治理：敏感词过滤、限频控制、权限校验
- 互动能力：点赞去重、计数回收与一致性处理

## 项目价值

- 适合作为全栈实战项目与课程作业展示
- 覆盖社区产品核心链路：内容、互动、治理
- 具备可扩展架构，便于二次开发与功能演进

## 快速开始

克隆项目后，按后端与前端目录分别安装依赖并启动服务。
